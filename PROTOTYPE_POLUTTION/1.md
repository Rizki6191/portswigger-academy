# Lab: Client-side prototype pollution via browser APIs

Challenge Description:
 This lab is vulnerable to DOM XSS via client-side prototype pollution. The website's developers have noticed a potential gadget and attempted to patch it. However, you can bypass the measures they've taken.

Goal:
 Find a source to add arbitrary properties to Object.prototype, identify a gadget property that executes JavaScript, then call alert().

Solution:
 English: Use __proto__ as a prototype pollution source, set a gadget property on Object.prototype, and trigger alert().
 Indonesian: Gunakan __proto__ sebagai sumber prototype pollution, atur properti gadget pada Object.prototype, dan jalankan alert().

Payload:

true    
/?=__proto__[value]=foo
