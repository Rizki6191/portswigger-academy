# Lab: Prototype pollution exploit payload

Challenge Description:
 This lab demonstrates a prototype pollution payload that uses __proto__ to set a gadget sequence property.

Goal:
 Understand how __proto__ can inject a property that triggers arbitrary execution.

Solution:
 English: Use __proto__.sequence to set the gadget property and execute alert(1)-.
 Indonesian: Gunakan __proto__.sequence untuk menetapkan properti gadget dan mengeksekusi alert(1)-.

Payload:
/?__proto__.sequence=alert(1)-
