# Lab: Prototype pollution bypass payload

Challenge Description:
 This lab demonstrates bypassing input filters using a crafted __pro__proto__to__ parameter.

Goal:
 Bypass filtering and set transport_url to data:,alert(1); via a crafted request parameter.

Solution:
 English: Use the __pro__proto__to__ parameter to bypass input validation and set transport_url to data:,alert(1);.
 Indonesian: Gunakan parameter __pro__proto__to__ untuk melewati validasi input dan mengatur transport_url ke data:,alert(1);.

Step:
1. Pertama cari prototype pollution source seperti `__proto__` lalu juga cari gadgetnya, disini gadget nya adalah `transport_url`
2. Setelahnya coba dengan `?__proto__[transport_url]=foo` dan cek dengan `console.log(Object.prototype.transport_url);` jika menghasilkan output foo berarti prototype pollution berhasil
3. Karena tujuan nya adalah memicu alert() yang berarti harus mencari pemicunya, disini sama seperti sebelumnya bisa dengan format data URI `data:[<mediatype>][;base64],<data>`, dan ini payloadnya `/?__proto__[transport_url]=data:,alert(1);` tetapi ini tidak berhasil dan ternyata `__proto__` di filter, ini bisa dibypass dengan `__pro__proto__to__` ini akan membypass filternya. payload akhir: `?__pro__proto__to__[transport_url]=data:,alert(1);
`
Payload:
