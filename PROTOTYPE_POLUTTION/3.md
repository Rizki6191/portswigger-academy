# Lab: DOM XSS via an alternative prototype pollution vector

Challenge Description:
 This lab demonstrates a prototype pollution payload that uses __proto__ to set a gadget sequence property.

Goal:
 Understand how __proto__ can inject a property that triggers arbitrary execution.

Solution:
 English: Use __proto__.sequence to set the gadget property and execute alert(1)-.
 Indonesian: Gunakan __proto__.sequence untuk menetapkan properti gadget dan mengeksekusi alert(1)-.

Step:
1. Pertama cari prototype pollution source seperti `__proto__` dan atur gadget properti misalnya disini yaitu `[value]` dengan valuenya yaitu foo, tetapi ternyata tidak menghasilkan foo karena terkena filter, cara lainnya bisa dengan `.value` yang akan menghasilkan foo. 
2. Selanjutnya cari gadget property dan disini didapatkan jika gadget nya adalah `sequence`
3. Sisanya sama seperti sebelumnya `/?__proto__.sequence=alert(1)`

Payload:
/?__proto__.sequence=alert(1)-
