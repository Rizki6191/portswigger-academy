# Lab: SQL injection attack querying the database type and version on MySQL and Microsoft

Challenge Description:
 This lab contains a SQL injection vulnerability in the product category filter. You can use a UNION attack to retrieve the results from an injected query.

Goal:
 Display the database version string.

Solution:
 Use a UNION SELECT payload that returns the version string in one of the result columns.

Payload:
```http
GET /filter?category='   HTTP/2
```

The response should contain the database version string.
