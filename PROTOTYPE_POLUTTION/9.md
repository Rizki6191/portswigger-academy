# Lab: Bypassing flawed input filters for server-side prototype pollution

Challenge Description:
 This payload abuses a flawed filter by using constructor.prototype to set isAdmin indirectly.

Goal:
 Achieve admin rights through server-side prototype pollution without direct __proto__ injection.

Solution:
 English: Use constructor.prototype.isAdmin=true to bypass flawed filters and set admin rights.
 Indonesian: Gunakan constructor.prototype.isAdmin=true untuk melewati filter yang cacat dan menetapkan hak admin.

Step:
1. Pertama sesuai deskripsi chall cari prototype pollution source seperti `__proto__`
2. Selanjutnya cari pemicu dimana ini bisa dijalankan yaitu di `POST /my-account/change-address` jika di capture dengan burpsuite dan send request, response nya akan menunjukkan `"isAdmin":false`
3. untuk mengubahnya menjadi true tidak bisa gunakan ini payload ini `"__proto__":{ "isAdmin":true}` karena di filter, ini bisa di bypass dengan `"constructor":{ "prototype":{ "isAdmin":true}}` sisanya masuk ke admin panel, dan delete user carlos

Payload:
"constructor":{ "prototype":{ "isAdmin":true}}
